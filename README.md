# 🚨 AI Smart Emergency Response System

An AI-powered web application that provides real-time emergency guidance using intelligent responses. Designed to assist users during critical situations with step-by-step instructions and safety advice.

---

## 🌐 Live Demo

* **Frontend (Vercel):** https://
smart-emergency-response-system-nine.vercel.app
* **Backend (Render):** https://smart-emergency-response-system-5o7u.onrender.com

---

## 🧠 Features

* 🚨 AI-based emergency response system
* 🧾 Structured guidance (steps, do’s & don’ts)
* 🎙️ Voice input support
* 🔊 Text-to-speech output
* 🆘 SOS alert simulation
* 📶 Offline fallback guidance
* ⚡ Fast and responsive UI
